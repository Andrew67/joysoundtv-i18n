# joysoundtv-i18n
WebExtension which translates the JoySound.TV KyokuNavi (Web UI)

Supported consoles:
* PlayStation 3

Supported target languages:
* en-US

Supported browsers:
* Firefox 57+

Supported device sizes:
* Desktop/tablet (width >= 768px)
